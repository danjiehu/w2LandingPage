# Sublime Text Shortcuts

#### Mac Keyboard Shortcuts

```
open a file in the current project         # cmd + p (or cmd + t)
paste with correct indentation             # cmd + shift + v
select word or its next occurrence         # cmd + d
select all occurences                      # ctrl + cmd + g
move a line upwards in a file              # ctrl + cmd + ↑
move a line downwards in a file            # ctrl + cmd + ↓
comment + uncomment lines                  # cmd + /
search in a file                           # cmd + f
search in the current project              # cmd + shift + f
add a package                              # cmd + shift + p
add indentation                            # tab
remove indentation                         # shift + tab
go to the next word                        # alt + →
go to the previous word                    # alt + ←
go to the end of line                      # cmd + →
go to the beginning of line                # cmd + ←
close the current tab                      # cmd + w
reopen the last tab closed                 # cmd + shift + t
navigate to the tab on the right           # cmd + alt + →
navigate to the tab on the left            # cmd + alt + ←
duplicate line                             # cmd + shift + d
cut line                                   # cmd + x
```

#### Linux Keyboard Shortcuts

```
open a file in the current project         # ctrl + p
paste with correct indentation             # ctrl + shift + v
select word or its next occurrence         # ctrl + d
select all occurences                      # alt + F3
move a line upwards in a file              # ctrl + shift + ↑
move a line downwards in a file            # ctrl + shift + ↓
comment + uncomment lines                  # ctrl + /
search in a file                           # ctrl + f
search in the current project              # ctrl + shift + f
add a package                              # ctrl + shift + p
add indentation                            # tab
remove indentation                         # shift + tab
go to the next word                        # alt + →
go to the previous word                    # alt + ←
go to the end of line                      # Home
go to the beginning of line                # End
close the current tab                      # ctrl + w
reopen the last tab closed                 # ctrl + shift + t
navigate to the tab on the right           # Page Down
navigate to the tab on the left            # Page Up
duplicate line                             # ctrl + shift + d
cut line                                   # ctrl + x
```
